# SmartFilter

**SmartFilter** – פתרונות סינון והגנה דיגיטלית

Next.js 14 production site for SmartFilter, Ashdod.

## Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Vercel** (deployment)

## Getting Started

```bash
npm install
npm run dev
```

## Deploy to Vercel

```bash
npm i -g vercel
vercel --prod
```

Or connect the GitHub repo at vercel.com and it deploys automatically.
